# Qwen3 Tiny Reference Fixtures

Golden fixtures for the Qwen3 differential (cross-validation) test.
Generated by a tiny Qwen3 model built with HuggingFace transformers
and converted via `weight_converter.py`.

## Regenerate

```bash
python3 Applications/CausalLM/res/qwen3/test/generate_qwen3_reference.py
```

Requires: `torch >= 2.0`, `transformers >= 4.51` (Qwen3 support).

After regeneration, commit the updated files in this directory.

## Files

| File | Description |
|---|---|
| `config.json` | Tiny Qwen3 model config (matches C++ `makeTinyQwen3Config`) |
| `generation_config.json` | Generation parameters |
| `nntr_config.json` | nntrainer runtime config |
| `tokenizer.json` | Tiny WordLevel tokenizer (vocab size 32) |
| `nntr_qwen3_tiny_fp32.bin` | FP32 weights in nntrainer binary format |
| `input_ids.json` | Fixed token IDs used for both HF and nntrainer inference |
| `reference_logits.json` | HF prefill last-token logits (32 floats) |
| `reference_tokens.json` | HF greedy token sequence |
| `meta.json` | Seed, tolerances, library versions used at generation time |
